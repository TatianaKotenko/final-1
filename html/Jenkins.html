Jenkins

  • build — запускает сборку и при необходимости ждёт её завершения.
  • clear-queue — очищает очередь сборки.
  • console — получает вывод консоли сборки.
  • connect-node — пересоединяет с узлом (узлами).
  • groovysh — запускает интерактивную оболочку Groovy.
  • help — выводит список всех доступных команд или подробное описание одной команды.
  • install-plugin — устанавливает плагин из файла, по URL или из центра обновлений.
